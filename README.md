# MidnightStudio OS

A premium, web-based desktop simulator built with React and Vite. Experience a nostalgic yet modern operating system interface directly in your browser.

## âœ¨ Features

- **Window Management**: Full window system with dragging, resizing, snapping, and focus management.
- **Desktop Environment**: Functional desktop icons, marquee selection, and custom context menus.
- **Built-in Apps**:
  - **Portfolio Suite**: About, Resume, Impact, and Contact apps.
  - **Terminal**: A retro-style interactive terminal.
  - **Music**: Integrated music player (Midnight Music).
  - **System Tools**: Trash, Changelog, and Feedback apps.
- **Immersive Design**: Dynamic backgrounds (including shooting stars), smooth animations, and a polished aesthetic.
- **Easter Eggs**: Try **Shift + Clicking** for a special cursor trail effect!
